# 3.发布需求接口

# 3.1 地址

baseUrl + /api/process/b2b/postwant

# 3.2 请求方式

POST

# 3.3 参数

参数 注释 是否必填 备注
mobile 手机号 是 -
connectName 联系人 是 -
provinceId 省份编码 是 如：370000000000
cityId 城市编码 是 -
districId 地区编码 是 -
provinceName 省份名称 是 -
cityName 城市名称 是 -
districName 地区名称 是 -
addressDetail 详细地址 是 -
proGroup 产品组 是 如 彩电,中央空调 (多个以逗号隔开)
proGroupCode 产品组编号 是 如 11,0 (多个以逗号隔开)
email 邮箱 是 -
companyName 公司名 是 -
memo 备注 否 -
code 验证码(用于自注册,不是非必填) 否 -
